## Usage

Press `s` to focus the search bar. Search queries can be prefixed with flags to change search engine (default is Google).

| Flag | Search Engine |
|------|---------------|
| `-g` | Google        |
| `-i` | Google Images |
| `-y` | Youtube       |
| `-j` | Jisho         |
| `-k` | Kotobank      |
| `-n` | Nyaa          |
| `-s` | Sukebei       |
| `-p` | Anime-Planet  |
| `-v` | VNDB          |
| `-a` | Arch Wiki     |
| `-w` | Wikipedia     |
